Venus Framework

Venus Framework是一个对Spring封装的应用框架,基于为了快速构建标准化、可扩展系统思想设计的超轻量级标准化中间件

使用方便

基于 XXXXXXXXXXXXX功能。

模块化开发

支持模块化开发，应用中的每一个模块都含有独立的 Spring 上下文，模块之间的调用通过本地服务的方式来完成。

健康检查

提供了一套可扩展的健康检查机制，可以帮助确定应用启动完毕后是否健康，是否可以对外提供服务。
产品功能
服务发布与引用
通过 JVM 服务的发布与引用，解决了隔离 模块的通信问题， 提供了三种方式给开发人员来发布 JVM 服务和引用，分别是： XML 方式、Annotation 方式及编程 API 方式 。
基于 Velocity 的 Spring 配置
的 Spring 配置文件基于 Velocity，根据用户的不同配置文件，将初始化不同的 Bean，提升灵活性。
生命周期管理
框架包含完整的生命周期管理，应用可以监听 应用或者 组件的生命周期事件。
Log4j2 支持
框架支持 Log4j2，实现日志的异步打印。同时， 框架基于 DRM 实现了动态调整日志级别功能。
扩展点
框架的模块是相互隔离的， 扩展点允许一个模块对其依赖的模块中定义的组件进行定制化。
健康检查
框架提供了一套可扩展的健康检查机制，可以帮助应用确定启动完毕后是否健康，是否可以对外提供服务。
