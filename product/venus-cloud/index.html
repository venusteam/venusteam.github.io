Venus Cloud

Venus Cloud 是分布式服务框架中的服务注册中心，目前同 IDC 内部的服务寻址都是通过 Confreg 来实现的，服务注册的粒度是接口+协议。

容量无限扩展

支持水平扩展，理论上无容量瓶颈。

秒级实时性

服务地址变更秒级推送。

容灾自恢复

服务端对网络故障有一定容忍性，网络故障恢复后集群自动恢复。
产品功能
服务治理
服务提供方在启动时会以“接口+协议”的方式注册到 Confreg。
API网关
服务消费方从 Confreg 获取服务提供方列表，并缓存到内存中，调用服务时采用点对点直连的方式。
负载均衡
由服务消费方实现负载均衡，服务消费方决定如何从服务提供方的IP列表中选取一个服务器。
失效转移
Confreg 会自动检测服务提供方的健康状态。如果服务提供方出现故障，Confreg 会自动将其从列表中剔除，并推送新的列表到服务消费方，服务消费方不会再调用到故障机器。
支持多种域名
支持标准内部域名、自定义域名(普通域名)，映射域名(Global 域名)。
