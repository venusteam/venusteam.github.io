Venus Boot

Venus Boot 是Venus Team基于 Spring Boot 的中间件轻量集成方案，与标准的 Spring Boot 工程无缝集成。其在完美集成最优中间件的前提下对用户提供了易用、统一的编程界面。

基于 Spring Boot

Venus Boot 底层是基于 Spring Boot 的，天生便可与 Spring Boot，Spring 工程无缝集成，大大减少了用户的迁移成本。

集成开源最优中间件

集成开源最优中间件，并提供统一易用的编程界面，用户只需引入所需中间件的 Starter 变能直接使用所需的中间件 。

原生云工程结构

满足原生云应用的十二个因素，具备快速开发，持续交付和部署服务，弹性扩展，故障隔离、自动恢复等特点。

统一易用的编程接口

每一个Venus Boot Starter都是独立可插拔的组件，对集成的中间件提供统一易用的编程接口，节约开发时间，和后期维护的成本。
产品功能
中间件的集成管理和自动配置
Venus Boot 提供中间件提供轻量级集成方案，即用户只需要引入对应中间件的 Starter ，Venus Boot 会自动导入所需的依赖并完成必要的配置。
XXX 模式
Venus Boot 提供 Embedded Venus 模式以便用户在 Spring 编程环境下使用Venus Team开发的中间件。
XXXXX
Venus Boot 集成日志埋点工具 Tracer，提供统一的中间件日志埋点和上下文 ID，通过一个统一的 ID，将上下游系统的调用关系串联起来。
多类型的部署模式
Venus Boot 支持将应用打包成 FAT JAR 包的方式进行运行，同时也支持在多种 Servlet 容器（包括 Tomcat，Jetty，Undertow）中运行。
应用日志和中间件日志的隔离能力
Venus Boot 通过使用应用引入的日志实现类来打印中间件的日志，Venus Boot 不依赖具体的日志实现，均面向 SLF4J 接口进行日志打印，并提供了日志隔离的能力。
多维度应用度量
提供多种度量维度实时监测应用程序的性能，能帮助更好的了解当前应用程序或者服务在线上的各种性能状态。
