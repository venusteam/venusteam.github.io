创建预发单

更新时间: 2017-2-21

创建预发单，可以支持多种形式的内容，如：本地文件， 本地targz包， 字符串buffer流，ossUrl地址。

将一个本地文件发布到应用服务器的目标目录

const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetDir: ’/home/admin/uiweb/site/home/screen’,
  file: path.join(__dirname, ’./fixtures/sample.txt’),
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

将一个本地targz包发布解压到应用服务器的目标目录

const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetDir: ’/home/admin/uiweb/site/home/screen’,
  targz: path.join(__dirname, ’./fixtures/sample.tgz’),
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

将oss上的一个targz包发布解压到应用服务器的目标目录

const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetDir: ’/home/admin/uiweb/site/home/screen’,
  ossUrl: ‘http://alipay-rmsdeploy-image.cn-hangzhou.alipay.aliyun-inc.com/rmsportal/scCkJsslgPOKYvh.tar.gz',
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

将字符内容发布到应用服务器的目标文件

const buffer = new Buffer("I’m buffer", "utf-8");



const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetFile: ’/home/admin/uiweb/site/home/screen/buffer.txt’,
  data: buffer,
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

发布到应用的开发环境

const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetDir: ’/home/admin/uiweb/site/home/screen’,
  file: path.join(__dirname, ’./fixtures/sample.txt’),
  targetEnv: ‘env’,
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

tagetEnv = ‘dev’ 将会发布到应用的线下服务器, 默认 targteEnv = ‘prod’ 发布到线上环境.

发布到指定IP的服务器

const params = {
  targetApps: [ ‘cmscenter’, ‘cmspromo’ ],
  targetDir: ’/home/admin/uiweb/site/home/screen’,
  servers: [ ‘10.244.38.168’, ‘10.244.38.169’ ]
  targetEnv: ‘env’,
  tag: ‘jssdk-test-1’,
};



yield filesync.create(params);

servers 参数如果提供的话，则优先使用 servers 参数提供的IP.
