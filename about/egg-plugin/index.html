Egg/Chair 插件

更新时间: 2017-5-12

我们强烈建议 Egg/Chair 系统的用户直接使用 Basement File 的插件。通过插件可以简单地和 Egg/Chair 框架高度整合。

配置插件

安装依赖

tnpm i @alipay/egg-basement –save

开启插件

config/plugin.js

exports.basement = {
  enable: true,
  package: ’@alipay/egg-basement’,
};

配置插件行为

打开: config/config.{env}.js

exports.basement = {
  // 单实例配置
  client: {
    appId: ‘your app id’,
    masterKey: ‘your master key’,
    env: ‘prod’, // 默认不用设置，如果是非线上环境使用线上环境的 basement 服务，需要设置成 prod
    // encryptPassword: false, // masterKey 是否加密，默认 false
  },



// 多实例配置
  // clients: {
  //   instance: {
  //     appId: ‘your app id’,
  //     masterKey: ‘your master key’,
  //     env: ‘prod’, // 默认不用设置，如果是非线上环境使用线上环境的 basement 服务，需要设置成 prod
  //     // encryptPassword: false, // masterKey 是否加密，默认 false
  //   },
  // },



// 创建实例时的默认配置，不论是单实例还是多实例或者是动态创建都会合并这份默认配置
  default: {
    env: ‘stable’,
    timeout: ‘3s’,
  },
};

使用方式

Controller 中使用

const FileClient = this.service.basement.file;

Service 中使用

const FileClient = ctx.service.basement.file;
